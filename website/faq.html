AntiCMSP Browser - FAQ

Respostas para as dúvidas mais frequentes sobre o navegador.

O que é o AntiCMSP Browser?

O AntiCMSP Browser é um navegador desenvolvido para facilitar a injeção de scripts nas plataformas de ensino, como CMSP, Alura, Leia SP, entre outros.

O navegador é seguro para uso?

O navegador foi projetado com funcionalidades úteis para os alunos. No entanto, o uso dos scripts adicionados pelo usuário via TamperTurkey ou qualquer outro meio deve ser feito com cautela. Deve ser evitado o uso do navegador como padrão, pois o navegador vem com a segurança desativada para que haja sucesso na injeção de scripts e manipulação de páginas.

Quais sistemas operacionais são compatíveis?

Atualmente, o AntiCMSP Browser funciona em Windows e Linux

Como instalar o AntiCMSP Browser?

Visite a página inicial e baixe uma das versões disponíveis.

Posso personalizar os scripts utilizados?

Os scripts presentes no menu lateral direito não podem ser editados ou personalizados diretamente, mas é possível adicionar e criar scripts para páginas utilizando o TamperTurkey.

Como injetar um script?

Os scripts podem ser acessados clicando nas 3 barras horizontais presentes no canto superior direito da interface do navegador, ao clicar irá aparecer um menu lateral que conterá os scripts disponíveis. (pode ser necessário rolar um pouco o menu para baixo) Para injetar o script basta clicar no script desejado.

Como injetar o TamperTurkey?

O TamperTurkey pode ser injetado por clicar em seu ícone no canto superior direito, ao lado esquerdo das 3 barras horizontais.
